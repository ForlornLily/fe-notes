# 简单数据类型/基本类型

simple data types/primitive types

- undefined
- null
- Number
- String
- Boolean
- Symbol (ES6 新增)
- BigInt (ES10 新增)

## null

null 是个空对象指针

如果定义的变量将来要赋值成对象，那么声明的时候最好给 null

## undefined

void xxx

void 后面跟任意表达式都一样, 比如`void 0`

旧版本的浏览器`undefined`不是保留字，可以当做普通变量被赋值。void 的目的就是返回 undefined
