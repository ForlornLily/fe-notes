# 转换规则总结

## 数据类型

比如[]转数字

首先[]调 valueOf，返回本身

再调 toString()返回""

""转数字是 0

## 加法

- 运算中其中一方为字符串，那么就会把另一方也转换为字符串

- 如果一方不是字符串或者数字，那么会将它转换为数字或者字符串

## 除了+号以外的算术运算

乘除取余，递增/减.一元加/减

都先转数字
